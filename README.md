# MemberRegisterDiscord 🤖

**MemberRegisterDiscord**, Discord sunucuları için geliştirilmiş, modern ve özelleştirilebilir bir üye kayıt sistemidir. Sunucunuza yeni katılan üyelerin kayıt işlemlerini otomatize eder ve moderasyon sürecini kolaylaştırır.

---

## 🚀 Özellikler

* **Otomotize Kayıt Sistemi:** Yeni gelen üyeleri karşılar ve kayıt rollerini atar.
* **Modüler Yapı:** `functions.js` ile ayrılmış mantık katmanı sayesinde kolayca geliştirilebilir.
* **Hoş Geldin Mesajları:** Kayıt kanalına özel karşılama mesajları gönderir.
* **Gelişmiş Komut Yapısı:** Moderatörler için hızlı kayıt komutları içerir.
* **Cloud Ready:** `Procfile` ve GitHub Actions entegrasyonu sayesinde Heroku, Railway gibi platformlarda 7/24 çalışmaya hazırdır.

---

## 🛠️ Kullanılan Teknolojiler

* **Node.js:** JavaScript çalışma ortamı.
* **Discord.js:** Discord API ile iletişim kurmak için kullanılan güçlü kütüphane.
* **JavaScript (ES6+):** Modern programlama standartları.
